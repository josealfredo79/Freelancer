1¿Qué es FreelancerEscrow?

FreelancerEscrow es un contrato inteligente en Stellar Soroban que permite:

💰 Pagos Seguros

Los fondos se retienen en el contrato hasta que se completan los hitos

📋 Gestión de Hitos

Divide el proyecto en entregas medibles y aprobadas

⚖️ Resolución de Disputas

Un árbitro neutral puede mediar en conflictos

🔒 Cancelación Segura

Devuelve fondos si el contrato se cancela

2Instalar Entorno de Desarrollo

1. Instalar Rust

# En Linux/macOS
$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
# or
$ sudo apt install rustc

# Verificar
$ rustc --version
rustc 1.75.0

2. Instalar wasm32 target

$ rustup target add wasm32-unknown-unknown

3. Instalar Stellar CLI

$ cargo install stellar-cli --locked
# o descargar binario
$ wget https://github.com/stellar/stellar-cli/releases/latest/download/stellar-cli-x86_64-unknown-linux-gnu.tar.gz
$ tar -xzf stellar-cli-x86_64-unknown-linux-gnu.tar.gz
$ mv stellar-cli /usr/local/bin/

4. Verificar Instalación

$ stellar --version
stellar 21.0.0

$ rustc --version
rustc 1.75.0

$ cargo --version
cargo 1.75.0

5. Clonar Proyecto

$ git clone https://github.com/tu-usuario/freelancers.git
$ cd freelancers
$ cargo build

8Ejemplo de Uso

// 1. Crear hitos
let hitos = vec![&env, create_hito(&env, 1, "desarrollo", 1000)];

// 2. Inicializar contrato
client.initialize(&empresa, &freelancer, &arbitro, &token, &hitos);

// 3. Depositar fondos
client.deposit(&1000);

// 4. Aprobar hito
client.approve_milestone(&1);

// 5. Liberar pago
client.release(&1);

// 6. Verificar estado
let escrow = client.query_escrow();
assert(escrow.estado == Estado::Completado);
