Instalar Entorno

1. Instalar Rust

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2. Instalar Stellar CLI

cargo install stellar-cli --locked

Compilar Contrato

1. Agregar al Cargo.toml

[lib]
crate-type = ["cdylib", "rlib"]

2. Instalar target WASM

rustup target add wasm32-unknown-unknown

3. Compilar

cargo build --release --target wasm32-unknown-unknown

4. Verificar archivo

ls target/wasm32-unknown-unknown/release/freelancer_escrow.wasm

Deploy en Stellar Testnet

1. Generar cuenta

stellar keys generate testuser

2. Obtener XLM de prueba

curl "https://friendbot.stellar.org/?addr=TU_DIRECCION"

3. Desplegar contrato

stellar contract deploy --wasm target/wasm32-unknown-unknown/release/freelancer_escrow.wasm --source testuser --network testnet
Importante: Copia el CONTRACT_ID que te devuelve el comando deploy.

Invocar Contrato

Reemplaza CONTRACT_ID con el ID que te dio el comando deploy.

1. Inicializar

stellar contract invoke --id CONTRACT_ID --source testuser --network testnet initialize --empresa "ADDR" --freelancer "ADDR" --arbitro "ADDR" --token "ADDR" --hitos '[{"id":1,"descripcion":"demo","monto":1000}]'

2. Depositar

stellar contract invoke --id CONTRACT_ID --source testuser --network testnet deposit --monto 1000

3. Aprobar hito

stellar contract invoke --id CONTRACT_ID --source testuser --network testnet approve_milestone --hito_id 1

4. Liberar pago

stellar contract invoke --id CONTRACT_ID --source testuser --network testnet release --hito_id 1

5. Consultar estado

stellar contract invoke --id CONTRACT_ID --source testuser --network testnet query_escrow

Funciones del Contrato

FunciónDescripción
initializeCrear contrato con empresa, freelancer, árbitro y hitos
depositDepositar monto total de los hitos
approve_milestoneAprobar hito completado
releaseLiberar pago del hito al freelancer
disputeAbrir disputa (empresa o freelancer)
resolveResolver disputa (árbitro)
cancelCancelar y devolver fondos
query_escrowConsultar estado del contrato
